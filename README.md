# KITE AI BOT

## PREREQUISITIES
Make sure Node.js is installed on your Linux system.

## FEATURES
1. Support Multiaccount
2. Support Proxy (Optional)
3. Auto Looping 24h

## INSTALLATION
1. Clone the Repository  

   ```sh
   git clone https://github.com/0xKii/kite-ai.git
   cd kite-ai
2. Install The required Dependencies

   ```sh
   npm install
3. Run Bot

   ```sh
   node index.js
